# LPOO1617_T4G11 - Final Project - Tron


## Setup and Installation
### Project
In Eclipse:
Import -> General -> Projects from Folder or Archive -> Select Directory -> Finish
To run select Class Launcher in Package GUI.

### App
Simply run the Tron.jar file.
